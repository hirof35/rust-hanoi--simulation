🦀 Rust Hanoi Tower GUI
Rustの軽量2Dゲームフレームワーク macroquad を使用して実装された、GUIベースの「ハノイの塔」パズルゲームです。
滑らかなアニメーション、効果音演出（設計）、そして自動解法AIアルゴリズムを搭載しています。

✨ 主な機能
直感的なGUI操作: 柱をクリックするだけのシンプルな「選択＆移動」システム。

滑らかなアニメーション: Lerp（線形補間） を用いた、円盤のヌルッとした滑らかな移動。

選択ハイライト: 現在どの円盤を掴んでいるかが一目でわかるビジュアルフィードバック（選択すると円盤が少し浮き上がります）。

不正移動の防止: 「小さい円盤の上に大きい円盤を置けない」という原作のルールをロジックレベルで厳格にガード。

自動解法（AI）モード: 再帰アルゴリズムを用いて、任意の盤面（初期状態）からゴールまでの最短手数を自動でシミュレーション再生する機能を搭載。

クリア判定: ゲームクリア時に画面が暗転し、リザルトエフェクトが表示されます。

🛠 動作環境・依存関係
言語: Rust (Edition 2021 以降推奨)

主なクレート:

macroquad = "0.4" (描画・ウィンドウ管理・入力・オーディオ)

🚀 クローンと実行方法
1. 前提条件
システムに Rust と cargo がインストールされている必要があります。未インストールの場合は rustup.rs を参照してください。

2. セットアップと実行
リポジトリをクローンするか、新規プロジェクトを作成してコードを配置します。

Bash
# 新規プロジェクトを作成する場合
cargo new rust_hanoi_tower --bin
cd rust_hanoi_tower
Cargo.toml の [dependencies] に以下を追加します：

Ini, TOML
[dependencies]
macroquad = "0.4"
src/main.rs にソースコードをコピーし、以下のコマンドで実行します：

Bash
cargo run --release
🎮 操作方法
手動プレイ:

動かしたい円盤がある「柱（または土台）」を左クリックして選択します。（選択されると円盤が浮き上がり、枠線が黄色になります）

移動させたい先の「柱」を再度左クリックすると、ルールに従って円盤が移動します。

選択をキャンセルしたい場合は、同じ柱をもう一度クリックします。

自動解法:

画面左上の [Auto Solve] ボタンをクリックすると、AIによる自動デモンストレーションが開始されます。

自動解法中はユーザーのクリック操作は一時的にロックされます。

🧠 コードのアーキテクチャ概要
データ構造: Rustのメモリ安全性を活かし、3本の柱を Vec<Vec<Disk>>（スタック構造）で表現。所有権システムにより、円盤の移動はメモリリークなく安全に行われます。

状態管理: C++（Siv3D）の -1 センチネル値を廃止し、Rust特有の Option<usize> 型（Some(index) / None）を用いることで安全なクリック判定を実現。

アニメーション: フレームレートに依存しない独立した描画座標（Vec2）を保持し、毎フレーム目標座標へ補間アプローチさせています。
